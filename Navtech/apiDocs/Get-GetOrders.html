Web API Help Page

GET api/order/GetOrders?pageIndex={pageIndex}&pageSize={pageSize}&isSortedByAscending={isSortedByAscending}

API is used to get the orders.

Parameters

NameDescription
pageIndex (Required) 
The page index for which records are to be fetched.
pageSize (Required) 
Number of records to be displayed in a page.
isSortedByAscending (optional) 
To get records from ascending or descending order.

On Failure

On failure, this method will return the following response based on parameters.

{
    "Response": "Success",
    "Message": "No orders found for the given input.",
    "Data": null,
    "ErrorCode": null
}
