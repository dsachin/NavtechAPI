Web API Help Page

Post api/Customer/CreateCustomer

Request Information

Request body formats

application/json,x-www-urlencoded

Sample: 
Name:Jack
Email:jack@qwe.com
Mobile:5453

Response Information

Response body formats

application/json, text/json

Sample: 
{
{
    "Response": "Success",
    "Message": "Customer added Successfully.",
    "Data": null,
    "ErrorCode": null
}
}

On Failure

On failure, this method will return the following response based on parameters. 
If the email is duplicate.
{
    "Response": "Error",
    "Message": "Duplicate Email Id.",
    "Data": null,
    "ErrorCode": "250"
}
                            
If inputs are invalid or not provided
{
    "Response": "Error",
    "Message": "Provided inputs are invalid.",
    "Data": null,
    "ErrorCode": "250"
}
